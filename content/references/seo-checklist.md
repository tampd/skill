# SEO Checklist — Optimization & Publishing

## Keyword Optimization
- [ ] Primary keyword: 1-2% density (tự nhiên, KHÔNG stuffing)
- [ ] Secondary keywords: 0.5-1% mỗi keyword
- [ ] LSI/NLP terms: rải đều, ngữ nghĩa đa dạng

## Readability
- [ ] Flesch-Kincaid: ≤ Grade 8 (dễ đọc)
- [ ] Câu ngắn: ≤ 20 từ/câu average
- [ ] Paragraph ngắn: ≤ 4 câu
- [ ] Subheadings: mỗi 200-300 words

## Links
- [ ] Internal links: ≥ 3 per 1000 words
- [ ] External links: ≥ 2 nguồn uy tín
- [ ] Anchor text đa dạng, tự nhiên
- [ ] Không broken links

## Technical SEO
- [ ] Title tag: ≤ 60 chars, chứa primary keyword
- [ ] Meta description: ≤ 155 chars, có CTA
- [ ] URL slug: ngắn, chứa primary keyword
- [ ] Single H1, proper H2→H3→H4 hierarchy
- [ ] Image alt text: mô tả + keyword tự nhiên

## Schema Markup
- [ ] Article schema (author, datePublished, dateModified)
- [ ] FAQ schema (nếu có section FAQ)
- [ ] HowTo schema (nếu là tutorial)
- [ ] Breadcrumb schema

## Pre-Publish
- [ ] Canonical URL set đúng
- [ ] Open Graph tags (og:title, og:description, og:image 1200x630)
- [ ] Twitter Card tags
- [ ] Sitemap updated
- [ ] Mobile-friendly formatting

## Post-Publish
- [ ] Submit URL qua Google Search Console
- [ ] Share lên social channels
- [ ] Internal link từ bài cũ → bài mới
- [ ] Monitor GSC: impressions, clicks, position (sau 2 tuần)
