# Content Engine — Multi-Platform Repurposing

> Workflow để chuyển đổi 1 bài viết gốc → nhiều platform format.

## Core Principle

```
1 Long-form Article → 5+ Platform Outputs
```

## Repurposing Matrix

| Source | → Platform | Format | Length | Hook Style |
|--------|-----------|--------|--------|------------|
| Blog post | Twitter/X | Thread | 10-15 tweets | Hot take / Controversial angle |
| Blog post | LinkedIn | Article | 500-800 words | Story lead / Professional insight |
| Blog post | Newsletter | Summary | 300-500 words | Curiosity gap / FOMO |
| Blog post | YouTube | Script outline | 5-10 min | Hook question / Promise |
| Blog post | Slide deck | Key points | 10-15 slides | Visual + 1 stat per slide |

## Twitter Thread Formula

```
Tweet 1: Hook — bold claim or question (get engagement)
Tweet 2-N: Key points — 1 idea per tweet, examples/data
Tweet N+1: Summary — "TL;DR: [3 bullet points]"
Tweet N+2: CTA — "Full article: [link]" + "Follow for more [topic]"
```

## LinkedIn Post Formula

```
Line 1: Hook — personal story or surprising stat
Line 2-3: Context — why this matters
Body: 3-5 key insights (short paragraphs)
CTA: Question to drive comments
Last line: Hashtags (3-5 relevant)
```

## Newsletter Formula

```
Subject line: [Curiosity gap] + [Benefit]
Opening: 1 sentence — what's new/important
Body: Top 3 insights from the article (summarized)
CTA: "Read the full analysis: [link]"
PS: Related recommendation or upcoming content
```

## Anti-Patterns

❌ Copy-paste the same text to all platforms
❌ Generic intros ("In this article, we'll explore...")
❌ Too many hashtags (looks spammy)
❌ No platform-specific formatting (threads need tweet-sized chunks)
❌ Missing CTA (always tell reader what to do next)
