## QUY TRÌNH 5 BƯỚC

### Step 1 — RESEARCH (Phân tích keyword & đối thủ)

```
1. Keyword Clustering:
   - Primary keyword: [từ khóa chính — volume cao nhất]
   - Secondary keywords: [3-5 từ khóa phụ liên quan]
   - LSI keywords: [5-10 từ khóa ngữ nghĩa — entities liên quan]
   - Long-tail: [2-3 câu hỏi cụ thể người dùng hay search]

2. Search Intent Analysis:
   | Keyword | Intent | Content Type |
   |---|---|---|
   | [keyword] | Informational / Transactional / Navigational / Commercial | Blog / Landing / Product |

3. SERP Analysis (Mental Model):
   - Top 10 đang viết về gì?
   - Content type phổ biến: listicle / how-to / review / comparison?
   - Content gap: chủ đề nào CHƯA AI cover đầy đủ?
   - Word count trung bình: [estimate]
   - Featured snippet: dạng nào (paragraph / list / table)?

4. E-E-A-T Assessment:
   - Experience: cần chia sẻ kinh nghiệm thực tế nào?
   - Expertise: cần chuyên môn gì để viết đáng tin?
   - Authoritativeness: nguồn nào cần cite?
   - Trustworthiness: dữ liệu/số liệu nào cần dẫn chứng?
```

### Step 3 — DRAFT (Viết nội dung GEO-optimized)

```
QUY TẮC VIẾT:

1. GEO Optimization (cho AI search engines):
   - Mở đầu mỗi section = 1 câu trả lời trực tiếp (AI extract được)
   - Dùng bullet points / numbered lists thường xuyên
   - Bảng so sánh khi có nhiều options
   - Cite nguồn: "Theo [Source], ..." (AI ưu tiên content có citations)
   - Structured data: FAQ, HowTo, Article schema

2. E-E-A-T Signals:
   - Chia sẻ kinh nghiệm thực tế: "Sau khi thử nghiệm X, chúng tôi thấy..."
   - Dữ liệu cụ thể: con số, thống kê, timeline
   - Author bio nếu có
   - Cập nhật thường xuyên: "Cập nhật [tháng/năm]"

3. Content Quality:
   - Mỗi paragraph ≤ 3-4 câu (mobile-friendly)
   - Active voice > passive voice
   - Transition words giữa paragraphs
   - Visual break: image/diagram/table mỗi 300 words
   - Avoid: fluff, filler, generic statements

4. Multimedia Suggestions:
   - [Vị trí 1]: [Mô tả image/infographic cần tạo]
   - [Vị trí 2]: [Mô tả diagram/chart]
   - [Vị trí 3]: [Mô tả video/screenshot]
```

### Step 4 — OPTIMIZE (Tinh chỉnh SEO)

```
CHECKLIST TỐI ƯU:

Keyword Density:
[ ] Primary keyword: 1-2% (tự nhiên, KHÔNG stuffing)
[ ] Secondary keywords: 0.5-1% mỗi keyword
[ ] LSI/NLP terms: rải đều, ngữ nghĩa đa dạng

Readability:
[ ] Flesch-Kincaid: ≤ Grade 8 (dễ đọc)
[ ] Câu ngắn: trung bình ≤ 20 từ/câu
[ ] Paragraph ngắn: ≤ 4 câu
[ ] Subheadings: mỗi 200-300 words

Links:
[ ] Internal links: ≥ 3 per 1000 words
[ ] External links: ≥ 2 nguồn uy tín
[ ] Anchor text đa dạng, tự nhiên
[ ] Không broken links

Technical:
[ ] Title tag: ≤ 60 chars, chứa primary keyword
[ ] Meta description: ≤ 155 chars, có CTA
[ ] URL slug: ngắn, chứa primary keyword
[ ] Single H1, proper H2→H3→H4 hierarchy
[ ] Image alt text: mô tả + keyword tự nhiên
```

### Step 5 — PUBLISH CHECKLIST

```
TRƯỚC KHI PUBLISH:

On-Page:
[ ] Title tag unique + compelling + ≤ 60 chars
[ ] Meta description có CTA + ≤ 155 chars
[ ] Chỉ 1 H1, heading hierarchy đúng
[ ] Images có alt text mô tả
[ ] Internal links ≥ 3
[ ] External links ≥ 2 (nguồn uy tín)
[ ] Mobile-friendly formatting

Schema Markup:
[ ] Article schema (author, datePublished, dateModified)
[ ] FAQ schema (nếu có section FAQ)
[ ] HowTo schema (nếu là tutorial)
[ ] Breadcrumb schema

Technical:
[ ] Canonical URL set đúng
[ ] Open Graph tags (og:title, og:description, og:image)
[ ] Twitter Card tags
[ ] Sitemap updated
[ ] Page speed OK (images optimized, lazy loading)

Post-Publish:
[ ] Submit URL qua Google Search Console
[ ] Share lên social channels
[ ] Internal link từ bài cũ → bài mới
[ ] Monitor GSC: impressions, clicks, position (sau 2 tuần)
```

## QUY TẮC

- **KHÔNG keyword stuffing** — tự nhiên, đọc mượt trước, SEO sau
- **LUÔN E-E-A-T** — kinh nghiệm thực > lý thuyết suông
- **GEO-first** — viết sao cho AI extract được câu trả lời trực tiếp
- **Content > Keywords** — nội dung giá trị thực sự > chạy theo keyword
- **Mobile-first** — paragraph ngắn, visual breaks, scannable
- **Cite sources** — mọi claim cần dẫn chứng hoặc data
- **Update regularly** — evergreen content cần cập nhật mỗi 3-6 tháng
